Growth of a Tree

Fall 2017

Processing programming language

The project was completed in the Design 37 Processing Coding class at UC Davis. The project would ideally inspire viewers to analyze on their own experiences and personal growth as well as reflect how they have contributed to society.

Ideation and impact

The project is centered around an apple tree that represents personal growth. The Ideation was inspired by Cedric Kiefer and Jennifer Steinkamp who used generative design to visualize concepts and data. A different tree generates each time the code is run.

The different types of apples (red, yellow, green) represent an individualâ€™s accomplishments, and the outliers (bananas, oranges, and potatoes) represent unwanted or unexpected experiences. The outliers are also larger in size than the regular apples. They are emphasized because people tend to think about their mistakes a lot more than their successes. The first image in the sequence is a generated tree trunk, independent from the final project, generated using arrays.
